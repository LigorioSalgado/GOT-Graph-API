# GOT GRAPH API

## Requirements
- Nodejs
- Express
- Mongodbd (with mongoose)
- Graphql & express-grapql

## How start
- Install dependencies:
    ```sh
        $ npm install
    ```
- run project;
     ```sh
        $ npm run  start
    ```
- Open you browser and  go to [localhost:3000/graphql](http://localhost:3000/graphql/)
